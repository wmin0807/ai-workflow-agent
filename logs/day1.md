## 💡 Java vs Python 函数对比

| 特性 | Java 写法 | Python 写法 | 我的理解 |
|------|----------|-------------|---------|
| 函数定义 | `public int add(int a, int b)` | `def add(a: int, b: int) -> int` | ... |
| 类型提示 | 强制，编译时检查 | 可选，运行时不强制 | ... |
| 文档注释 | `/** ... */` | `""" ... """` | ... |
| 测试写法 | `@Test public void testAdd()` | `assert add(2,3)==5` | 这种写法确实简单、便捷 |
